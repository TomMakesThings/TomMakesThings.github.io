Joe's Pyramid Solver (2021)

Joeâ€™s pyramid is a puzzle in which every stone is marked with a different one or two digit positive number. Where a stone rests on two others, its number is the sum of the numbers marked on the two stones on which it rests. The challenge is to find the value of the top stone X.

PseudobuLk and Single cell Mixture models for Investigating Differential expression (PLASMID) (2021)

During an internship, I worked in collaboration with UK DIR Imperial College London to benchmark pseudobulk, pseudoreplicate and mixture model methods for identifying differentially expressed genes in microglia.

My contribution was creating PLASMID - a parallelizable Snakemake pipeline to run multiple differential expression methods and their variations (41 in total), given input scRNA-seq data. Supported methods include: DESeq2, edgeR, limma, Welch and Wilcoxon t-tests, MAST and NEBULA.

After running each method, log fold-change and adjusted p-values are outputted and the results visualised through custom built interactive graphs. These allow a user to visually identify up and down regulated genes and compare the methods against an expected ground truth.

Semi-Supervised Learning with TCNs for ECG Classification (2021)

I managed a degree project in which we experimented using a new semi-supervised learning approach to identify arrhythmia (a type of heart condition) from ECG time-series data.

First we applied k-means clustering to create new labels for each ECG, and then used them to train a temporal convolutional network (TCN). Then we employed transfer learning to create a new network for classifying the original arrhythmia labels.

Though unfortunately transferring network weights did not make a significant difference compared to training the network without, we still consider the project a success: Our final model achieved 97% accuracy demonstrating TCNs are much more suitable for time series ECGs than CNNs or LSTMs as used in previous literature.
